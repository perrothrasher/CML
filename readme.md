# Proyecto de Machine Learning

Este proyecto utiliza análisis de datos utilizando un dataset meteorologico.
Utiliza técnicas de clustering para agrupar los datos.

## Archivos
- `cml.yaml`:
- `readme.txt`: Descripción del proyecto
- `requeriments.txt`: Dependencias necesarias para ejecturar el proyecto
- `data.csv`: Dataset meteorologico a utilizar
- `train.py`: Código principal para el análisis y modelado de datos
